┌────────────────────────────────────────────────────────────────────────────┐
│ 🏗️ ARCHITECTURE                                                            │
└────────────────────────────────────────────────────────────────────────────┘

COMPONENT COMMUNICATION
───────────────────────
  Document Detail Page
         ↓
  TanStack Query (Smart Polling)
    - Document (3s if processing)
    - Job (3s if processing)
    - Chapters (when completed)
         ↓
  Conditional Rendering
    - Processing → Timeline
    - Completed → Player + Chapters
    - Failed → Error + Retry
         ↓
  Audio Player ←→ Chapter Navigation
    ↓                ↓
  localStorage    Custom Events


STATE MANAGEMENT
────────────────
  TanStack Query:
    • ['document', documentId]
    • ['processing-job', documentId]
    • ['chapters', documentId]

  localStorage:
    • sonoro_player_{docId}_position
    • sonoro_player_{docId}_speed

  Local State:
    • isPlaying, currentTime, volume
    • currentChapter, playbackSpeed

POLLING STRATEGY
────────────────

  • Stop polling when status is completed/failed
